# ATXLab-750G

Projeto de testador profissional para fontes ATX de até 750 W, com carga eletrônica multicanal, medição AC/DC, proteção independente de firmware e suporte à validação de eficiência compatível com 80 PLUS Gold.

## Estrutura

- `docs/arquitetura/` — requisitos, arquitetura e matriz de testes
- `docs/esquematico/` — documentação SCH-750G-001
- `hardware/kicad/` — captura KiCad do módulo 12 V / 10 A
- `releases/` — pacotes ZIP consolidados

## Status

Engenharia preliminar. Não liberado para produção ou uso sem validação elétrica, térmica, mecânica e de segurança.
